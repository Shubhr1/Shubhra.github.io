SCHEMATIC:

So what we will be needing for vehicle tracking system is as follows:

1.GPS Receiver module with an antenna to receive GPS data and then send the GPS data to an arduino board.

2. Arduino must be programmed such that it converts the GPS data into understandable latitude and longitude values.

3. GSM module must pick this data from arduino board and sent it to your phone via msgbr>
4. Finally, the data received in your phone must be used to plot or trace the route of the vehicle that you are tracking.

INTERFACING ARDUINO UNO:

Install arduino software by using www.arduino.cc

Connect arduino board to laptop using USB-URT cable.LED on board starts glowing.Go to device manager-->expand the ports column-->you can now look at the port assigned to arduino board.In case it shows warning sign--right click on it and click on update driver software-->browse on my computer.Refresh device manager u can now see the warning sign has gone.

Your set up is complete now.Open the software now you will see different options under FILE,TOOLs,etc..

Under Tools click on Boards-->Arduino UNO, Ports-->Port assigned(in my case it was PORT8).In arduino software programs written are called sketches.

ARDUINO PROGRAM:

In #include we include the library which we wanna use.Every sketch needs 2 void type function.

1. set up() :runs only for one time.We initialise system here.

2. loop() :runs for infinite time.All our logic resides here.

On board LED is assigned pin 13.

PROGRAM FOR LED TO ALWAYS REMAIN OFF:

Int ledPin = 13;

void set up()

{

PinMode(ledPin,OUTPUT);

}

void loop()

{

digitalWrite(ledPin,LOW); //we state 2 variables in this mode.The level which can be HIGH on state and LOW offstate.

}

Next we will save the file by clicking on Save As and then click on compile.It will check for errors in the program written..Click on tool to check the board and port used,it must be the same we used earlier.Finally,Upload the code to arduino by connecting the board using the /USB-URT cable.



You can also write another program for blinking LED.

PROGRAM FOR BLINKING LED

Int ledPin = 13;

void set up()

{

PinMode(ledPin,OUTPUT);

}

void loop()

{

digitalWrite(ledPin,HIGH); //we state 2 variables in this mode.The level which can be HIGH on state and LOW offstate.

delay(1000); //giving a delay of 1000milli seconds.

digitalWrite(ledPin,LOW);

delay(1000);

}

Save the file.Compile And upload it to Arduino UNO board.

INTERFACING GPS MODULE TO ARDUINO

Connections of GPS module with Arduino UNO board are as follows:

GPS MODULE ARDUINO UNO
GND GND
5V pin 5V pin
TX pin (i.e. transmitter pin) TX pin
RX pin (i.e. receiver pin) RX pin


Connect Reset and GND of Arduino UNO with each other.When we ground the reset pin the Arduino works as dumb USB serial connrctor.

This is done to use Arduino UNO just as a communication device between the laptop and GPS receiver module.

Connect the antenna of GPS module and tighten the screw.

Antenna should be kept closer to the room exterior as possible to establish proper lock with the GPS satellite.

So the interfacing has finished now.

Connect the USB-URT cable to laptop and arduino.You will observe that LED starts glowing.

To read the data--Open the arduino software with a blank sketch.Go to Tools-->Boards-->ARDUINO Uno.Tools-->PORT-->COM8(whatever is assigned in ur laptop).Tools-->Serial monitor

You will observe a similar data.this data is called synamio data.The data will have sentences which will begin with '$'sign called sentence identifier.It will have following set of data:

GPGGA, GPGSA, GPGSV, GLGSV.Our focus in this project will be on GPGGA.Each value has its specific meaning:

1st value-TIME in UTS format-HHMMSS.ms

2nd and 3rd value-LATITUDE in degree and minutes(DMS format) and direction N OR S respectively.

4th and 5th value-LONGITUDE in degree and minutes(DMS format) and direction E OR W respectively.

6th value-GPS Quality Indicator:0-no fix, 1-GPS fix.

7th value-number of satellites in use.NOTE:For locating a point we need 4satellites to lock with the GPS Module

We need these 7values only for our project.

Google maps take values of latitude and longitude in decimal degrees.

We can now use these values in google map application to locate the points of GPS module.For this open Google map in your browser and first input LATITUDE and then LONGITUDE.

+ve for N,-ve for S,+ve for E,-ve for W.You can now successfully read the location.yayyy!!!

INTERFACING GMS MODULE TO ARDUINO

Insert the SIM card in the module(at the back of module you will observe a slot provided for sim).

Connections of GMS module with Arduino UNO board are as follows:

GMS MODULE ARDUINO UNO
GND GND(present before pin 13)
TX pin (i.e. transmitter pin) Digital pin 0(RX)
RX pin (i.e. receiver pin) Digital pin 1(TX)


Power them both.Power GSM module using 12V adaptor.You will see power LED and Ring LED glows on GSM module.Long press the power button to establish 'network connections'.Once you release it NETWORK LED starts blinking rapidly.As soon as SIM card establishes network connectivity with mobile network, LED starts blinking at lower frequencyYou can now try to call the GSM module.You can hear the ringing tone on ur phone and will observe RING Led starts glowing.Now you have a working mobile device.
