In this project we will be using Lilypad arduino to check the pulse of a person.We will also be using pulse sensors,7segment display module and battery.

CONNECTING AND DETECTING LILYPAD ARDUINO

We need Lilypad Arduino,female pin connector,FTDI connector,USB Cable

In lilpad arduino:first two pins are GND, 3rd pin is for VCC, 4th pin is for RX(receiver), 5th pin is for TX(transmitter), 6th pin is VTR.

Connections of lilpad arduino and FTDi connector:

FTDI CONNECTOR LILYPAD ARDUINO0
GND GND(2ndpin)
VCC pin VCC pin
TX pin (i.e. transmitter pin) RX pin
RX pin (i.e. receiver pin) TX pin
DTR pin VTR pin


For power connect mini USB pin to FTDI and other pin to laptop.Led on arduino starts blinking and led on FTDI glows.So u are done with the interfacing of both.

SCHEMATIC

Pulse sensor: Identifies the pulse and measure the heart rate.

Lilypad arduino:Reads the data from pulse sensor at regular intervals and send data to display module.

TM1637 Display module:; Displays the heart beat rate.

Connections of lilpad arduino and FTDi connector:

Component 1 Pin Pin description Pin Pin Description Component 2
Pulse sensor + 5V positive + 5V positive Lilypad arduino
- Ground - Ground
S Signal Pin A0 Analog pin
TM 1637 Display module VCC 5V positive + 5V positive Lilypad Arduino
GND Ground - Ground
DIO Data I/O 9 Digital pin
CLK Clock 10 Digital pin

ARDUINO LIBRARY INSTALLATION

Download 2 header files for pulse sensor and TM1637 Display.Unzip it using WinRaR.Copy these 2 files and paste ithem in library of Arduino.

For library Go to My computers->Local drive C->Program filesx86->Arduino->library(paste).

If still the files doesn't show in the folder reopen the folder,it will refresh it and you will be able to see the files you copied.

INTERFACING PULSE SENSOR TO ARDUINO

As connections are already mentioned in the table above.We will connect + pin of lilypad to VCC of pulse sensor(middlepin).-pin of lilpad to GND of pulse sensor(right most).a0 pin to signal pin of sensor(left most).

Connect the USB to laptop.Led starts glowing of arduino board as well as pulse sensor.OPen the program u have written in Arduino RDE.Go to Tools-->Serial monitor.

Put your finger on pulse sensor.You will observe the readings on serial monitor.

DESIGN SKETCH FOR TM1637 DISPLAY MODULE

const byte CLK = 10;

const byte DIO = 9;

int i;

SevenSegmentTM1637 display(CLK,DIO);
void setup()

{ Serial.begin(9600);

display.begin();

display.setBacklight(100); }

void loop()

{ Serial.print("Good");

display.print("Good");

delay(1000);

display.clear();

for(i=0;i<=20;i++);

{ Serial.print(i);

display.print(i);

delay(200);
}
display.print("DONE");

Serial.print("DONE");

delay(1000);

display.clear();

delay(1000);

}
Save the file.Verify and compile and upload it to lilypad.Do check the board and port opted in tools.

INTERFACING AND TESTING TM1637 DISPLAY MODULE

Connections are already stated above in the table.Connect GND of TM1637 to +pin of lilypad.VCC to -ve pin. DIO pin to digital pin 9 of Lilypad. CLK to digital pin 10 of Lilypad.

Connect USB to laptop.Open the file created of TM1637 display module.Led of arduino will glow and display module will show the msg.It means the connections are done properly.

FABRICATION SCHEMATIC


1.Sew the lilypad on the back side of glove.

2.Sew the pulse sensor on the index finger of palm side of glove.Please ensure that the sensing side is facing outwards.

3.Sew the TM1637 DisplayModule on thumb of the back side of glove.
Note: Sewing must be done by non-conductive thread.Also sew the battery holder at the bottom of glove on back side.
Using the conductive thread the connections are done as shown.Refer the table above for connections.
Make sure no 2 threads touch each other while doing the connections, otherwise it may short and glove will catch the spark.
